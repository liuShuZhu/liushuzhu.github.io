发表于 2016-11-04

concat()

  • 描述:
    连接两个或更多的数组，返回新生成的数组。
  • 语法以及参数:
    arrayObject.concat( arrayX, ... ) 参数可以为数组，可以为具体的值。
    首先对于原始数组来说，它会把原始数组复制一份，然后将参数中的元素放入复制的数组中并返回该数组。如果参数为数组的话，它会将数组拆开，将拆开后的元素放入数组中，需要注意的是该方法内部只会将参数为数组的参数拆开，如果被拆开的数组中仍有元素为数组，则方法内部不会将该数组保留。

join()

    将数组中的元素用规定字符连接为字符串。
  • 语法以及参数:
    arrayObject.join( separator ) 如果不传参数或者参数值为 undefined(IE8以上) ，默认以 , 连接。如果数组中的某一项值为 null 、 undefined 则系统内部会将其默认为空字符串表示。

pop()

    arrayObject.pop() pop删除后原数组的长度将会减1，如果数组空数组则长度不变，返回undefined。 该方法直接对原数组进行操作。

reverse()

  • 描述:
    颠倒数组中元素的位置。
  • 语法以及参数:
    arrayObject.reverse() 该方法内部通过替换的方式改变原来的数组中元素的顺序，不创建新的数组，返回改变后的数组。

shift()

  • 描述:
    删除数组的第一个元素，并且返会该元素的值。
  • 语法以及参数:
    arrayObject.shift() 直接对数组进行修改，如果数组为空数组则返回undefined。
  • 实例:

unshift()

  • 描述:
    向数组的开头添加一个或多个元素,返回新数组的长度。
  • 语法以及参数:
    arrayObject.unshift( newelement1, newelement2,...., newelementX ) 直接对数组进行修改。需要注意的是当调用 unshift 方法时，如果传入的参数较多它会一次性插入数组的开头，并且按照传入的参数的顺序插入数组中，即参数的第一项为数组为数组的第一项。
  • 实例:

slice()

  • 描述:
    从某个已有的数组中返回选定的数组。
  • 语法以及参数:
    arrayObject.slice(start,end) start 是选取的开始的位置，如果 start 的值为负值，则从数组末尾开始查找起始的元素。end 是一个可选项，如果有就是截取的终点的位置，但截取的数组不包括该元素，如果为负值则从数组的末尾确定终点的位置，如果没有值则默认截取到数组末尾。需要注意的是如果起始的位置大于终点的位置则返回 [];
